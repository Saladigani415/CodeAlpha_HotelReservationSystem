# Hotel Reservation System

## Project Description

Hotel Reservation System is a Java console-based application that helps users to search, book, and manage hotel room reservations.  
The application allows users to view available rooms, select room categories (Standard, Deluxe, Suite), make reservations, and cancel bookings.

## Features

- Search available hotel rooms
- Add room categories (Standard, Deluxe, Suite)
- Display room details and prices
- Book hotel rooms
- Cancel reservations
- Manage customer reservation details
- Display reservation summary report

## Technologies Used

- Java
- ArrayList
- Scanner
- Object-Oriented Programming (OOP)
- VS Code
